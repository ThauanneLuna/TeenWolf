# 🌑 Sobre o Projeto

O **Teen Wolf Fãs — Beacon Hills** foi desenvolvido como um projeto temático inspirado na série da MTV.

A proposta do site é criar uma experiência imersiva para fãs da série, reunindo:

- 🐺 Personagens
- 🌕 Curiosidades
- 🧩 Quizzes
- 🎬 Bastidores
- 🗺️ Mapa interativo
- 📖 Diário de Beacon Hills
- ✨ Easter eggs

Tudo com um visual cinematográfico, misterioso e moderno.
